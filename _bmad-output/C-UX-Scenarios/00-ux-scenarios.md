## Scenarios

### [01: Maya's Tenant Standup](01-mayas-tenant-standup/01-mayas-tenant-standup.md)
**Persona:** Maya — hard tenancy proof + keys/audit  
**Pages:** Sign-in, Accounts & Tenants, API keys, Audit, Health & ops  
**User Value:** Two Tenants live; key issued once; audit + health green  
**Business Value:** CAP-2/8; isolation demonstrable

### [02: Ken's Knowledge Studio](02-kens-knowledge-studio/02-kens-knowledge-studio.md)

**Pages:** Documents, Document detail, Knowledge Studio, Thread/Retrieve, Trace  
**User Value:** Evidence artifacts + cost; readable hops  
**Business Value:** CAP-3/4/5; time-to-proof
