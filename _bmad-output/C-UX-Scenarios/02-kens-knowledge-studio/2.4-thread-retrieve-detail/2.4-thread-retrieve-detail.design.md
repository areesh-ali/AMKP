## Overview

**Page Purpose:** Deliver the climax — ranked Evidence + citations + CostEstimate; PreferCorrectness / insufficient_evidence first-class.

**Success Criteria:** Ken can cite a Document ID from an Evidence card and see estimated cost without leaving the thread.

## Sections

### Evidence list
- Ranked **EvidenceCard**: score · mono documentId · location · snippet  
- Teal 3px left accent  
- Keyboard ↑↓ select; Enter opens Artifacts drawer  

### Result header
- `requestId` mono muted  
- **CostPill** always adjacent  
- Outcome chip: `evidence` | `insufficient_evidence`  

### Optional gloss
- Short grounded summary *below* Evidence — secondary, never alone  

### Actions
- “Open Trace” → 2.5  
- “View Document” → 2.2
