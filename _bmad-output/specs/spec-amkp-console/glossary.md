# AMKP Console — Glossary

Plane terms inherit from `../spec-amkp/glossary.md`. Brand is **AMKP**.

| Term | Meaning |
| --- | --- |
| **AMKP** | Brand and enterprise knowledge plane (better RAG, hard tenancy, guarded agentic). |
| **AMKP Console** | First-party web product for humans to operate the plane. Does not replace SDK/MCP. |
| **Plane** | API + worker implementing `SPEC-amkp`. |
| **Platform Admin** | Console role for Accounts, Tenants, keys, audit, health/ops. |
| **Tenant Operator** | Console role bound to one Active Tenant. |
| **Active Tenant** | Tenant selected in session; all Operator actions scoped to it. |
| **DX kit** | SDK, MCP, OpenAPI, reference app — remain under AMKP. |
